# Proposal: PowerPlaza Product Catalog & Quote Inquiry Site

## Problem Statement

파워프라자(볼커파워)의 전력 변환 제품(DC-DC 컨버터, AC-DC 컨버터, EV 부품)을 국내에서 소개하고 판매하기 위한 전용 웹사이트가 필요합니다. 기존 파워프라자 사이트(powerplaza.net)에서 판매 중인 모든 제품군을 체계적으로 소개하고, 고객이 원하는 제품에 대해 견적 문의를 할 수 있는 시스템이 필요합니다.

## Proposed Solution

Next.js 15 기반의 제품 카탈로그 사이트를 구축합니다. 주요 기능:

1. **제품 카탈로그**: 3대 카테고리(DC-DC, AC-DC, EV Component) 아래 30+ 시리즈의 제품을 카탈로그 형태로 보여줍니다. 각 제품은 이미지, 기술 스펙, 가격, 인증마크, 데이터시트 다운로드 링크를 포함합니다.
2. **견적 문의 시스템**: 고객이 제품을 선택하고 수량과 함께 견적 문의를 할 수 있습니다. 회원가입 없이 이름/회사/연락처/이메일만 입력하면 됩니다. 문의가 접수되면 관리자에게 이메일 알림이 발송됩니다.
3. **관리자(어드민) 패널**: 이메일/비밀번호 로그인 방식. 초기 관리자 계정 자동 생성. 제품 CRUD, 카테고리 관리, 견적 문의 목록/상세 조회, 대시보드.
4. **다국어 지원(i18n)**: 한국어 + 영어 동시 지원.

## Scope

### In-Scope

- 메인 페이지 (히어로 배너 + 카테고리별 인기 제품)
- 제품 목록 페이지 (카테고리/시리즈별 필터링)
- 제품 상세 페이지 (이미지, 스펙 테이블, 가격, 인증마크, 데이터시트 링크)
- 견적 문의 폼 (제품 선택 + 고객 정보)
- 어드민 로그인 (better-auth, 이메일/비밀번호)
- 어드민 대시보드 (문의 현황 요약)
- 어드민 제품 CRUD (추가/수정/삭제)
- 어드민 카테고리/시리즈 관리
- 어드민 견적 문의 목록/상세
- 견적 문의 이메일 알림 (관리자 설정 가능)
- 한국어/영어 다국어 지원
- 푸터에 회사 정보 (수정 가능)
- 초기 제품 데이터 시딩 (파워프라자 사이트 기반, 최대한 많이)
- 파워프라자 제품 이미지 URL 활용

### Out-of-Scope

- 온라인 결제/장바구니 (견적 문의만)
- 고객 회원가입/로그인
- 제품 리뷰/댓글
- 블로그/뉴스
- 검색 엔진 최적화 (추후 작업)
- 소셜 미디어 연동

## Success Criteria

- [ ] 3대 카테고리 아래 모든 시리즈 제품이 조회 가능
- [ ] 고객이 제품 선택 후 견적 문의를 제출할 수 있음
- [ ] 관리자가 로그인하여 제품 CRUD, 문의 관리 가능
- [ ] 한국어/영어 전환 가능
- [ ] 모바일 반응형 디자인
- [ ] 초기 시딩 데이터에 최소 50개 이상의 제품 포함
